
Shell sort -
Best case O(nlogn)
Worst case  O(n^2)

Usage: sort.jar is located in the main folder of the project
Files created by the program will be located in the res folder of the project

Once navigated through the correct folder on a cmd window
the sort jar file can be run

Usage: sort.java
java -jar sort.jar -f<file> -t<type> -s<sort>

Where 
<file> is the filename of the file in the res folder
<type> is the compare type for the shapes (a,h, or v)
<sort> is the sorting algorithm to be used

the order of the arguments passed in will not matter.
